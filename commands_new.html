Module COMPAT

Description:

This module provides command for controlling memtable.

Commands:

  • LISTMAINTAINANCE
  • SAVE
  • RELOAD

LISTMAINTAINANCE

Command LISTMAINTAINANCE
Available since 1.3.5.3
Redis compatible N
Mutable n/a
Module COMPAT

Usage:

LISTMAINTAINANCE

Description:

Run crontab list maintainance. For example, crontab list maintainance may fsync binlogs.
Very useful if crontab list maintainance is disabled.

Return type:

OK

SAVE

Command SAVE
Available since 1.0.0
Redis compatible Y
Mutable n/a
Module COMPAT

Usage:

SAVE

Description:

Flushes memtable to the disk.
Reloads the disktable(s) from the disk.
Tthis is no-op on immutable servers.

Return type:

OK

Aliases:

  • BGSAVE

See Also:

  • RELOAD

RELOAD

Command RELOAD
Available since 1.0.0
Redis compatible N
Mutable n/a
Module COMPAT

Usage:

RELOAD

Description:

Reloads the disktable(s) from the disk.
This command is used when disktable(s) are updated or replaced from an external process.

Return type:

OK
